Ripple.js

Proper Material Design ripple effect for the web, that wouldn't interfere your elements' DOM.

Github & Docs

Easy to use

Hold to see it fill your container
Multiple ripples are possible 

            var buttons = document.querySelectorAll(".demo-1 button");

            ripple.bindTo(buttons);

Even easier


            ripple.watch(".demo-2 button");

Any shape, any size

You can edit nextBtnProps on the right 
            
              var nextBtnProps = {
                "width": "300px",
                "height": "600px",
                "border": "solid 8px #000",
                "borderRadius": "20px",
                "background": "#333"
              }

              ripple.watch(".demo-3 button");

Customizable

You can edit rippleBind.factory.rippleProps on the right 

            var rippleBind = ripple.watch(".demo-4 button");

            rippleBind.factory.rippleProps = {
                "color": "#2196F3", //background color (all your css supported)
                "opacity": 0.25,
                "borderRadius": 20, //makes it a bit faster on slow cpu
                "borderWidth": "auto", //or value in px
                "zIndex": 999, //riplle mask is positioned above target
                "transitionDuration": 675, //duration may varie depending on container size
                "timingFunction": "cubic-bezier(.4, 0, .42, 1)", //timing function
                "constant": false // enlarging speed is constant
            }
